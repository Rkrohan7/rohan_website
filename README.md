# Rohan Kadam — Portfolio

A modern, responsive personal portfolio website for **Rohan Kadam**, Java Full Stack Developer.

## ✨ Features
- Responsive design (desktop, tablet, mobile)
- Dark / light theme toggle
- Animated hero with typing effect & stat counters
- Projects carousel/slider with autoplay, arrows, dots & swipe
- Contact form with email delivery via [Web3Forms](https://web3forms.com)
- Zero build step — pure HTML, CSS & JavaScript

## 📁 Structure
```
index.html    # markup
styles.css    # styling & theme
script.js     # interactions (theme, slider, contact form)
```

## 🚀 Deploy
This is a static site — host the folder anywhere:
- **Netlify Drop:** drag this folder onto https://app.netlify.com/drop
- **GitHub Pages:** push to a repo → Settings → Pages → deploy from `main`
- **Vercel:** import the repo at https://vercel.com

No build command, no dependencies. Just serve `index.html`.

## 🛠 Tech
HTML5 · CSS3 · Vanilla JavaScript · Web3Forms
